# Neplance
A Freelancing Platform for Nepal , Neplance.


# Using the server 

Terminal : 
git clone https://github.com/dibyaadhikaree/Neplance.git  \n

cd server

# Installing all the modules required
npm i 

# Create a .env file inside /Server

PORT = 3000 \n
DB_URL = //create a local database on your mongoDb compass and paste the connection string \n
JWT_SECRET  = //put a random password of your choice \n
JWT_EXPIRES = 24 \n


# Running the Backend

npm start
